# Gracias AI Logo Design

## Files

- `gracias-ai-logomark.svg` - Icon-only version for favicons, app icons
- `gracias-ai-wordmark.svg` - Full logo with "Gracias AI" text

## Design Concept

The logo combines three core elements:

1. **Shield Shape** - Represents security, compliance, and trust. Uses iOS-style rounded corners to tie into the App Store ecosystem.

2. **Neural Circuit Pattern** - A stylized AI brain with circuit nodes and connections, representing the LLM-powered intelligence behind the product.

3. **Checkmark** - A subtle accent at the bottom representing the audit/compliance verification that Gracias AI provides.

## Color Palette

| Color  | Hex      | Usage                    |
|--------|----------|--------------------------|
| Purple | `#a855f7`| Primary brand color      |
| Blue   | `#3b82f6`| Secondary / gradient    |
| White  | `#ffffff`| Text and icon elements   |

The gradient flows from purple (top-left) to blue (bottom-right), creating a modern, dynamic feel.

## Usage

The logomark works well at small sizes (favicon, app icon) while the wordmark is ideal for headers, documentation, and marketing materials.

### Light Background
Use the default gradient version.

### Dark Background
Use the default version - the white circuit pattern and checkmark maintain excellent visibility.
