# Gracias AI Logo

A modern, minimalist logo design for Gracias AI App Store Policy Auditor.

## Design Concept

The logo combines three key elements:

1. **Shield Shape** - Represents security, compliance, and trust
2. **AI Neural Network Nodes** - Connected dots symbolizing artificial intelligence and intelligent analysis
3. **App Store Inspiration** - Rounded square overlay hints at iOS/App Store connection

## Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Purple | `#a855f7` | Main gradient start |
| Secondary Blue | `#3b82f6` | Main gradient end |
| White | `#ffffff` | Text/icons on dark backgrounds |

## Files Included

| File | Description | Background |
|------|-------------|------------|
| `logomark.svg` | Icon only | Light backgrounds |
| `logomark-dark.svg` | Icon only | Dark backgrounds |
| `wordmark.svg` | Icon + "Gracias AI" text | Light backgrounds |
| `wordmark-dark.svg` | Icon + "Gracias AI" text | Dark backgrounds |

## Usage Guidelines

### Light Backgrounds
Use `logomark.svg` and `wordmark.svg`

### Dark Backgrounds
Use `logomark-dark.svg` and `wordmark-dark.svg`

### Minimum Size
- Logomark: 32x32 pixels
- Wordmark: 200x67 pixels

### Clear Space
Maintain padding equal to 10% of the logo width on all sides.

## Design Principles

1. **Simple & Modern** - Clean geometric shapes with smooth gradients
2. **Versatile** - Works on both light and dark backgrounds
3. **Scalable** - Vector format ensures crisp rendering at any size
4. **Meaningful** - Each element represents a core value of the product
